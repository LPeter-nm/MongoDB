# Instruções
Clone o Repositório:<br>
`git clone https://github.com/LPeter-nm/MongoDB.git`<br>

Entre na pasta da aplicação:<br>
`$ cd MongoDB`<br>

Instale as Dependências:<br>
`npm install`<br>
`yarn install`<br>

Inicie o Servidor de Desenvolvimento:<br>
`npm run dev`<br>
